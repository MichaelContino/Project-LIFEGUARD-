# 🚨 Project Lifeguard

### *When seconds count, the AED needs to actually be there.*

---

## The Problem

Every year, thousands of people experience cardiac arrest near a publicly listed AED — and the device isn't there.

Not missing. Not broken. Just **wrong in the database.**

- ⏱️ **\< 90 seconds** — the window to act before permanent brain damage sets in  
- 📍 **\~40%** of AEDs in public registries are unverified or outdated  
- 💔 **10%** survival rate without an AED on scene

The gap between *listed* and *actually there* is costing lives.

---

## The Solution

**Project Lifeguard** is a real-time, crowdsourced AED verification layer.

It tells you — before you need it — whether the nearest AED is present, accessible, and functional.

### Three things it does:

|  |  |
| :---- | :---- |
| 📍 **Locate** | Find every AED within range, mapped in real-time |
| ✅ **Verify** | Community confirms: Is it there? Is it accessible? |
| 🚨 **Alert** | Flag expired, missing, or inaccessible units instantly |

---

## How It Works

01\. User opens app → GPS locates them → nearest AEDs appear on map

02\. Community verifies → passersby confirm AED presence during daily routines

03\. Data gets flagged → unverified or expired units surfaced to facility managers

04\. Registry updates → municipalities receive clean, current data

The flywheel: more users → better verification → cleaner data → more trust → more lives saved.

---

## Who This Helps

- **Bystanders** — locate a working AED in the critical first 2 minutes  
- **EMS Dispatchers** — direct callers to verified AEDs, not guesses  
- **Municipalities** — close the liability gap between their registry and reality  
- **Hospitals & Health Orgs** — prove community AED coverage to accreditors  
- **Facility Managers** — automatic compliance alerts before inspections  
- **Researchers** — real-world cardiac arrest geography and response data

## Where We Are

Project Lifeguard is in active early development and validation.

**Current phase:** Field audit — physically verifying AED accuracy within 5 miles of Highland Park, IL.

**Decision rule:** If 30%+ of listed AEDs fail verification → the product shifts from finder to full verification layer.
